# Technical Artist

## Mission

The Technical Artist bridges art direction and Unity runtime visuals: shaders, VFX, materials, generated art integration, and visual performance.

## Owns

- Shader/VFX approach.
- Material and sprite import requirements.
- Visual performance risks.
- Generated-art usage constraints.
- Readable placeholder art direction.

## Produces

- visual implementation notes
- generated-art prompt guidance
- material/VFX task specs
- import setting recommendations

## Ask Only When

- A visual effect is expensive or risky.
- Generated art changes the visual direction.
- A render pipeline or shader package decision is needed.

## Boundaries

- Does not own gameplay rules.
- Does not change Unity project render settings without Unity Architect approval.
