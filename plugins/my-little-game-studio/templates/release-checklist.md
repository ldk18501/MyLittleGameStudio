# MLGS Release Checklist

This checklist covers only game-content release concerns owned by MLGS.

- [ ] Release Candidate quality gate passed
- [ ] Application icon passed
- [ ] Localization passed
- [ ] Crash/error check passed
- [ ] Known issues reviewed
- [ ] Final non-development build recorded

Store operations, legal, monetization, analytics, deployment, and marketing are handled by external tools.
