## Rules

- Teach through the real production scene, UI, input, data, and save path.
- Do not count a text-only instruction sheet or developer-guided demo as onboarding.
- Every required mechanic has a teach, practice, verify, and recovery beat.
- Tutorial state is persisted or safely reconstructible.
- New-player comprehension must be playtested without developer guidance before Alpha approval.

## Evidence plan

- Vertical Slice: first-session path works with final-look UI and art.
- Content Complete: all release-scope tutorial beats are implemented and integrated.
- Alpha: a new player completes the intended first-session goal without developer guidance.
